# 가계부 웹앱

Google 로그인 기반의 개인 가계부 PWA. Firebase Firestore에 데이터를 저장하며 홈 화면에 추가해 앱처럼 사용할 수 있다.

## 비즈니스 로직

### 급여일
- 매월 24일 지급
- 24일이 주말 또는 공휴일이면 **직전 영업일**에 입금

### 카드 결제일
- 매월 25일 결제
- 25일이 주말 또는 공휴일이면 **다음 영업일**에 결제

### 카드 청구기간
- **전월 11일 ~ 당월 10일** 사용금액이 당월 25일(또는 다음 영업일)에 결제됨
- 예: 4월 25일 결제 = 3월 11일 ~ 4월 10일 사용분

### 급여월 단위
- 이번달 급여일 ~ 다음달 급여일 전날까지를 한 급여월로 처리
- 라벨은 급여일의 **다음 달** 기준으로 표시
- 예: 3월 24일(화) 급여 → 3/24~4/23이 한 급여월 → 라벨: "2026년 4월"

### 월 고정지출
실수령 급여에서 고정항목을 차감한 나머지가 **주식 투자 가능액**.
항목 및 금액은 `src/App.jsx`의 `DEDUCTIONS` 배열에서 수정.

### 카드 예산
- 한도는 `src/App.jsx`의 `BUDGET` 값으로 설정
- 현재 청구기간 내 사용금액 대비 남은 금액, 남은 일수, 일 예산 표시

### 공휴일
- 한국 공휴일을 코드에 하드코딩 (`HOLIDAYS` Set)
- 현재 **2025~2027년** 등록되어 있음
- 추가 필요 시 `src/App.jsx`의 `HOLIDAYS`에 `"YYYY-MM-DD"` 형식으로 추가

### SMS 문자 파싱 (자동 분석)
지원 형식:
- **NH농협 위카드**: `[NH농협] MM/DD HH:MM 위카드(체크) 승인 금액원 가맹점 잔액 금액원`
- **신한카드**: `신한카드(번호)승인 이름 금액원(일시불)MM/DD HH:MM 가맹점  누적금액원`
- **NH카드 멀티라인**: 카드명/이름/금액/날짜/가맹점이 줄바꿈으로 구분된 형식
- **이체**: `MM/DD 금액원 출금/입금 내용`

### 월 네비게이션
- 헤더의 ◀ ▶ 버튼으로 급여월 이동
- 현재 달이 아니면 **"이번 달로 돌아가기"** 버튼 표시
- 과거/미래 달 조회 시: "남은 일수" → "총 일수", "일 예산" → "일 평균 지출"로 전환

## 환경 설정

`.env` 파일을 프로젝트 루트에 생성:

```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

## 실행

```bash
npm install
npm run dev
```
